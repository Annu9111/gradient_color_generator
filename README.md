# 🎨 Gradient Color Generator

A modern and interactive Gradient Color Generator built using **HTML**, **CSS**, and **JavaScript**. Generate beautiful random CSS linear gradients with a single click and instantly copy the generated CSS code to your clipboard.

---

## 💡 How It Works

1. Click either color button.
2. A random HEX color is generated.
3. The page background updates with a new linear gradient.
4. The CSS gradient code is displayed.
5. Click the code box to copy the CSS instantly.

---

## 📷 Example Output

```css
background-image: linear-gradient(to right, #42A5F5, #7E57C2);
```
